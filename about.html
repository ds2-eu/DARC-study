About DS2 and DARC

This research is conducted within the EU DataSpace DataShare 2.0 project (DS2), which works to improve trustworthy data sharing capabilities across sectors by co-creating and piloting a modular and secure software ecosystem for data space interoperability.

These modules, collectively known as the Intersector DataSpace Toolkit, help data space participants:

  • Promote and search for data offers
  • Create trust and agreements
  • Share data between data spaces

DARC (Discovery, Assessment, Recommend, Configuration) is an intelligent conversational agent developed by DS2 consortium member ATC for choosing the right software tools to connect and interact with data spaces using the DS2 software ecosystem.

This study is aimed at companies, organizations, the scientific/research community, IT professionals, and public administrators who may be interested in interacting with the DS2 intelligent conversational agent.

Through DARC, users can:

  • Receive information, recommendations, and guidance for configuring the optimal pipeline of DS2 modules
  • Discover and configure software from the DS2 ecosystem
  • Search and understand available data spaces
  • Understand what information those data spaces include
  • Determine whether that information is relevant to their case
  • Learn how that information can be used for their purposes
